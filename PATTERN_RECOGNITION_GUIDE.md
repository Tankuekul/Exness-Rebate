## 🎯 SWING PATTERNS (18 รูปแบบหลัก)

### 🟢 **BULLISH PATTERNS (ซื้อ BUY)**

#### 1. **LH + LL → HH + LH** 🟢🟢 (แม่นมาก!)
```
     HH (4) ← เป้าหมายต่อไป
    /  \
   /    LH (3)
  /        
LH (1)   LL (2) ← Entry: ซื้อหลังเกิด HH
  \      /
   \    /
    ----
```
- **สถานการณ์:** ตลาด downtrend อ่อน → กลับขาขึ้น
- **การเข้า:** ซื้อเมื่อเกิด HH หลัง LL
- **เป้าหมายต่อไป:** HH ต่อเนื่อง (Bullish Continuation)
- **ความมั่นใจ:** 85-90%

#### 2. **LH + HL → HH + LH** 🟢 (แม่น)
```
       HH (3) ← Confirm!
      /  \
     /    LH (4)
    /
LH (1)
    \
     HL (2) ← ⚡ Early Entry!
```
- **สถานการณ์:** ตลาดสร้าง HL (Higher Low) หลัง LH
- **การเข้า (Early Entry):** ⚡ ซื้อทันทีที่เกิด HL (เร็ว!)
- **การเข้า (Normal):** รอให้เกิด HH confirm
- **เป้าหมายต่อไป:** Bullish Trend
- **ความมั่นใจ:** 80-85%

#### 3. **HH + LL → LH + HL** 🟢🟢 (BEAR TRAP - ซื้อ!)
```
HH (1) --------→ Target!
  \           ↗
   \        HL (4) ← 🟢 BUY!
    \      /
     LH (3)
      \  /
    LL (2) ← ❌ TRAP! (Liquidity Sweep)
```
- **สถานการณ์:** ตลาดทำ Fake Breakdown (LL) หลัง HH
- **การเข้า:** ซื้อเมื่อราคากลับมาสร้าง LH+HL
- **เหตุผล:** ผู้เล่นใหญ่ล้างตลาด (Liquidity Sweep) แล้วผลักขึ้น
- **เป้าหมายต่อไป:** กลับไป Test HH เดิม หรือสร้าง HH ใหม่
- **ความมั่นใจ:** 90%+

### 🔴 **BEARISH PATTERNS (ขาย SELL)**

#### 8. **LL + HH → HL + LH** 🔴🔴 (BULL TRAP - ขาย!)
- **สถานการณ์:** ตลาดทำ Fake Breakout (HH) หลัง LL
- **การเข้า:** ขายเมื่อราคาสร้าง HL+LH
```
     LH (2) ← ⚡ Early SELL! (Risky)
      /  \?
     /    HL? or LL?
LL (1)
```
- **เหตุผล:** ผู้เล่นใหญ่ล้าง BSL (Buy-Side Liquidity) แล้วดันลง
- **เป้าหมายต่อไป:** กลับไป Test LL เดิม หรือสร้าง LL ใหม่
- **ความมั่นใจ:** 90%+

#### 9. **HH + HL → LH** ⚠️ (Reversal Warning)
- **สถานการณ์:** Uptrend เริ่มอ่อน
- **การเข้า:** ขายเมื่อเกิด LH
- **เป้าหมายต่อไป:** รอ LL confirm → Downtrend
- **ความมั่นใจ:** 70-75%

```
   HH (2) ← ⚡ Early Entry!
   /
HH (1)
   \
    HL (3) ← Confirm
```
```
LH (1)
    \
     LH (2)
       \
        LL (3) ← 🔴 SELL!
          \
           ↘ Downtrend
```
#### 10. **LL + LH → LL** 🔴🔴 (Failed Reversal)
```
- **สถานการณ์:** ตลาดพยายามกลับขึ้น (LH) แต่ล้มเหลว → LL ใหม่
- **การเข้า:** ขายเมื่อเกิด LL ใหม่

     LH (2) ← Failed!
      /  \
     /    \
LL (1)    LL (3) ← 🔴 SELL!
            \
             ↘ Downtrend
```

- **เป้าหมายต่อไป:** Downtrend ต่อเนื่อง
- **ความมั่นใจ:** 85%

#### 11. **LL + LH (Early Entry)** 🔴⚠️ (Risky!)
- **สถานการณ์:** Downtrend → พยายามกลับขึ้น
- **การเข้า (Early):** ⚡ ขายทันทีที่เกิด LH (ก่อน HL confirm)
```
    LH ---- LH ---- LH (Resistance)
     \      /\      /
      \    /  \    /
       \  /    \  /
    HL ---- HL ---- HL (Support)
    
    ⚠️ รอ Breakout!
```
- **ความเสี่ยง:** สูง (อาจกลับขาจริง)
- **เป้าหมายต่อไป:** รอดูว่าจะเป็น LL ใหม่ หรือ HL
- **ความมั่นใจ:** 60-65%

#### 12. **HH + HH → HL** 🟡 (Overbought Reversal)
```
- **สถานการณ์:** Uptrend แรงมาก (HH ซ้ำ) → เริ่มอ่อน
- **การเข้า (Early Entry):** ⚡ ขายทันทีที่เกิด HH ตัวที่ 2 (เร็ว!)
- **การเข้า (Normal):** รอให้เกิด HL confirm

   HH (2) ← ⚡ Early Entry!
   /
HH (1)
   \
    HL (3) ← Confirm
```

- **เป้าหมายต่อไป:** ตลาด Sideways หรือ Reversal
- **ความมั่นใจ:** 65-70%

#### 13. **LH + LH → LL** 🔴🔴 (Bearish Strength - แม่นมาก!)
- **สถานการณ์:** Downtrend แข็งแกร่ง (LH ซ้ำ)
- **การเข้า:** ขายเมื่อเกิด LL
- **เป้าหมายต่อไป:** Downtrend ต่อเนื่อง
```

    /|

  /  Previous High

- **ความมั่นใจ:** 85-90%

   \  |

## 📊 BOS/ChoCh COMBINATION PATTERNS

### 🔵 **BOS (Break of Structure) Patterns**
```
    ┌─────────┐ ← Resistance (Flat)
    │    ↗ Breakout! 🟢
   /│   /
  / │  /
 /  │ /
/___│/
HL  HL ← Support (Rising)
```

#### 16. **SSL Sweep + Bullish BOS** 🟢🟢
```
- **สถานการณ์:** ราคาล้าง SSL (Sell-Side Liquidity) แล้วกลับขึ้นหลุด Structure
- **การเข้า:** ซื้อหลัง BOS confirm (ราคาหลุดแนว Swing High)

         ↗ Target
        /
    HH (BOS!) ← 🟢 BUY after Break
    /|
   / |
  /  Previous High
 /   |
HL   |
 \   |
  LL (SSL Sweep) ← Liquidity Grab
```

- **เป้าหมายต่อไป:** Uptrend
- **ความมั่นใจ:** 85%+

#### 17. **BSL Sweep + Bearish BOS** 🔴🔴
- **สถานการณ์:** ราคาล้าง BSL (Buy-Side Liquidity) แล้วกลับลงทะลุ Structure
```
LH  LH ← Resistance (Falling)
\___│\
 \  │ \
  \ │  \
   \│   \
    │    ↘ Breakdown! 🔴
    └─────────┘ ← Support (Flat)
```
- **การเข้า:** ขายหลัง BOS confirm (ราคาทะลุแนว Swing Low)
- **เป้าหมายต่อไป:** Downtrend
- **ความมั่นใจ:** 85%+

### 🔄 **ChoCh (Change of Character) Patterns**

#### 18. **Bullish ChoCh (HL break LL)** 🟢🟢
```
- **สถานการณ์:** Downtrend กลับขา → ราคาทำ HL แล้วหลุดแนว LL เดิม
- **การเข้า:** ซื้อหลัง ChoCh confirm

       ↗ Uptrend
      /
   HL (ChoCh!) ← 🟢 BUY
   /|
  / |─── Previous LL (Break!)
 /  |
/   |
   LL
```

- **เป้าหมายต่อไป:** Reversal to Uptrend
- **ความมั่นใจ:** 80-85%

#### 19. **Bearish ChoCh (LH break HH)** 🔴🔴
- **สถานการณ์:** Uptrend กลับขา → ราคาทำ LH แล้วทะลุแนว HH เดิม
- **การเข้า:** ขายหลัง ChoCh confirm
```
  Left    HEAD   Right
Shoulder    │   Shoulder
   │       HH       │
   HL      │\      HL
   │\     │ \     /│
   │ \    │  \   / │
   │  \   │   \ /  │
───┴───\──┴────X───┴─── Neckline
        \ │     \🔴 SELL!
         \│      ↘ Target
```
```
HH ═══════ HH (Failed!)
 \         /
  \   HL  /
   \  │  /
    \ │ /
─────\│/────── Support
      X 🔴 SELL!
      ↘ Target
```
- **เป้าหมายต่อไป:** Reversal to Downtrend
- **ความมั่นใจ:** 80-85%

## 📐 CLASSIC CHART PATTERNS

#### 22. **Ascending Triangle** 📐🟢
```
- **สถานการณ์:** HL+HL ต่อเนื่อง + แนวต้าน Flat
- **การเข้า:** ซื้อเมื่อหลุดแนวต้าน

        ↗ Breakout! 🟢
       /
      / (Wall หมดแล้ว)
─────X─────── Iceberg Level 🧊
     ⚡⚡⚡ (Failed attempts)
    ╱ ╲╱ ╲╱
   ╱  High Volume!
  ╱
```
- **เป้าหมายต่อไป:** Uptrend (เท่ากับความสูง Triangle)
- **ความมั่นใจ:** 75-80%

#### 23. **Descending Triangle** 📐🔴
```
- **สถานการณ์:** LH+LH ต่อเนื่อง + แนวรับ Flat
- **การเข้า:** ขายเมื่อทะลุแนวรับ

LH  LH ← Resistance (Falling)
\___│\
 \  │ \
  \ │  \
   \│   \
    │    ↘ Breakdown! 🔴 SELL!
    └─────────┘ ← Support (Flat)
         \
          ↘ Target (ความสูง Triangle)
```

- **เป้าหมายต่อไป:** Downtrend (เท่ากับความสูง Triangle)
- **ความมั่นใจ:** 75-80%

#### 24. **Symmetrical Triangle** 📐🟡
- **สถานการณ์:** HL+LH แคบลง (Consolidation)
- **การเข้า:** รอ Breakout ชัดเจน
- **ความเสี่ยง:** ไม่ทราบทิศล่วงหน้า
```
           ↗ Target (2x Range)
          /
    ┌────X──── Breakout! 🟢
    │   /│
LH──┤  / │ Retest Entry
    │ /  │
    │/   │
HL──┴────┘
    Range
    
Volume: ███████ (1.5x Spike!)
ATR:    ▲▲▲▲▲▲▲ (1.3x Expand!)
```
- **ความมั่นใจ:** 60-65%

#### 25. **Head & Shoulders** 👤🔴
```
- **สถานการณ์:** HH (Head) + HL ต่ำกว่า Shoulder
- **การเข้า:** ขายเมื่อทะลุ Neckline

  Left    HEAD   Right
Shoulder    │   Shoulder
   │       HH       │
   HL      │\      HL
   │\     │ \     /│
   │ \    │  \   / │
   │  \   │   \ /  │
───┴───\──┴────X───┴─── Neckline
        \ │     \🔴 SELL!
         \│      ↘ Target
```
- **เป้าหมายต่อไป:** Downtrend
- **ความมั่นใจ:** 80%+

#### 26. **Inverse Head & Shoulders** 👤🟢
```
- **สถานการณ์:** LL (Head) + LH สูงกว่า Shoulder
- **การเข้า:** ซื้อเมื่อหลุด Neckline

- **เป้าหมายต่อไป:** Uptrend
- **ความมั่นใจ:** 80%+
